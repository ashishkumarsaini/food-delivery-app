This project is a mobile application built using:

The app should prioritize:

- Clean UI

- Mobile-first interactions

# Design Guidelines

## General Design Direction

The application currently has:

- No finalized UI design
- A defined color palette
- Reference designs that will be shared later

The generated UI should:

- Feel modern and premium
- Use soft spacing and rounded corners
- Follow current mobile UI trends
- Maintain visual consistency across screens
- Be optimized for Android and iOS

The design inspiration should loosely follow the provided references while still keeping the UI original and assignment-appropriate.

# Color Palette

The color palette will be provided by the user. References: src/constants/theme.ts

Rules:

- Use the palette consistently
- Maintain accessible contrast ratios
- Avoid random colors outside the palette
- Use neutral backgrounds when necessary
